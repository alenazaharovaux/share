# Story Pitch

Pitch structure, angle development, and templates for daily news, features, investigations, op-eds, and freelance queries.

## Installation

```bash
cp -r story-pitch ~/.claude/skills/
```

No configuration needed — works out of the box.

## Works well with

- [ai-writing-detox](../ai-writing-detox/) — removes AI patterns from your pitch
- [newsroom-style](../newsroom-style/) — AP Style conventions

Based on [story-pitch](https://github.com/jamditis/claude-skills-journalism) by [Joe Amditis](https://github.com/jamditis) (MIT license).
