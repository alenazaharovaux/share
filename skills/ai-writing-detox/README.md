# AI Writing Detox

Eliminate AI-generated writing patterns that erode reader trust. Catches banned words, choppy dramatic fragments, corporate jargon, and fake enthusiasm.

## Installation

**macOS / Linux:**
```bash
cp -r ai-writing-detox ~/.claude/skills/
```

**Windows:**
```powershell
xcopy ai-writing-detox %USERPROFILE%\.claude\skills\ai-writing-detox\ /E /I
```

## What it covers

- **Banned words** — delve, leverage, robust, seamless, etc. with alternatives
- **Banned phrases** — throat-clearing, empty hedges, AI enthusiasm, corporate buzzwords
- **Banned rhythm patterns** — the #1 AI tell: choppy dramatic fragments
- **Substitution table** — plain-language replacements
- **Self-check checklist** — run before publishing

No configuration needed — works out of the box.

## Works well with

- [newsroom-style](../newsroom-style/) — AP Style and editorial conventions
