# Newsroom Style

AP Style and newsroom conventions for journalism writing. Numbers, titles, attribution, dates, formatting, ledes.

## Installation

```bash
cp -r newsroom-style ~/.claude/skills/
```

No configuration needed — works out of the box.

## Works well with

- [ai-writing-detox](../ai-writing-detox/) — removes AI-generated writing patterns

## Credits

Based on [newsroom-style](https://github.com/jamditis/claude-skills-journalism) by [Joe Amditis](https://github.com/jamditis) (MIT license).
