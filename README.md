# Share

Shared knowledge, tools, and skills for Claude Code.

## Skills

Ready-to-use skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Copy any skill folder to `~/.claude/skills/` to install.

| Skill | Description |
|-------|-------------|
| [triage-finding](skills/triage-finding/) | Quickly assess links, posts, articles, and YouTube videos — explains what it is, maps to your projects, recommends action |
| [tool-scout](skills/tool-scout/) | Find the right tools for your tasks — services, MCP servers, AI models, no-code, libraries. Searches for both established and new tools |
| [skill-creator](skills/skill-creator/) | Create, test, and deploy Claude Code skills using TDD principles. Extracted from [superpowers](https://github.com/anthropics/claude-code-plugins) by Jesse Vincent (MIT) |
| [ai-writing-detox](skills/ai-writing-detox/) | Eliminate AI-generated writing patterns — banned words, choppy fragments, corporate jargon, fake enthusiasm. Based on [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
| [newsroom-style](skills/newsroom-style/) | AP Style and newsroom conventions — numbers, titles, attribution, dates, ledes, formatting. Based on [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
| [story-pitch](skills/story-pitch/) | Pitch structure, angle development, templates for daily news, features, investigations, op-eds, freelance queries. Based on [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
| [newsletter-publishing](skills/newsletter-publishing/) | Email newsletter workflows — content strategy, HTML templates, subscriber management, deliverability, analytics. From [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
| [academic-writing](skills/academic-writing/) | Research methodology, paper structure (IMRaD), literature reviews, grant proposals, peer review, publishing strategy. From [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |

# Share (RU)

Общие знания, инструменты и скиллы для Claude Code.

## Скиллы

Готовые скиллы для [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Чтобы установить, скопируй папку скилла в `~/.claude/skills/`.

| Скилл | Описание |
|-------|----------|
| [triage-finding](skills/triage-finding/) | Быстро оценивает ссылки, посты, статьи и YouTube-видео — объясняет суть, сопоставляет с проектами, рекомендует действие |
| [tool-scout](skills/tool-scout/) | Находит инструменты для задач проекта — сервисы, MCP-серверы, нейронки, no-code, библиотеки. Ищет устоявшиеся и новые |
| [skill-creator](skills/skill-creator/) | Создание, тестирование и деплой скиллов Claude Code по принципам TDD. Извлечён из [superpowers](https://github.com/anthropics/claude-code-plugins) от Jesse Vincent (MIT) |
| [ai-writing-detox](skills/ai-writing-detox/) | Убирает паттерны ИИ из текста — запрещённые слова, рубленые фрагменты, корпоративный жаргон, фальшивый энтузиазм. На основе [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
| [newsroom-style](skills/newsroom-style/) | Стандарты AP Style и редакционные конвенции — числа, титулы, атрибуция, даты, лиды, форматирование. На основе [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
| [story-pitch](skills/story-pitch/) | Структура питча, развитие угла, шаблоны для новостей, фичеров, расследований, опедов, фриланс-запросов. На основе [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
| [newsletter-publishing](skills/newsletter-publishing/) | Рассылки — контент-стратегия, HTML-шаблоны, управление подписчиками, доставляемость, аналитика. Из [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
| [academic-writing](skills/academic-writing/) | Методология исследований, структура статьи (IMRaD), литобзоры, гранты, рецензирование, стратегия публикации. Из [jamditis/claude-skills-journalism](https://github.com/jamditis/claude-skills-journalism) (MIT) |
